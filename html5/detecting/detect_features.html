Supported HTML 5 Features

HTML 5 features have been checked for in this browser, and are summarized as follows:

  • Canvas supported
  • Canvas Text supported
  • Video supported
  • Video OGG format supported
  • Video H264 format supported
  • Local Storage supported
  • Web Workers supported
  • Offline Application supported
  • Geo Location supported
  • Microdata Formats supported

HTML 5 Input Fields Types and Form Features

The following HTML 5 input types have been checked for in this browser, and are summarized as follows:

  • Search supported
  • Number supported
  • Range supported
  • Color supported
  • Telephone supported
  • URL supported
  • Email supported
  • Date supported
  • Month supported
  • Week supported
  • Time supported
  • DateTime supported
  • DateTime-Local supported
  • Place holder in forms supported
  • Auto focus in forms supported
