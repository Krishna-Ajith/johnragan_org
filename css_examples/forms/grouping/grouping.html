To make this work, I set the labels to float left with a width of 20%. I had each label and field contained within a div, and set clear: left on the div due to the floating field inside.

For the access key, I set the accesskey property on the input to "f", "l" and so forth. I then used a span with a text-decoration of underline to highlight it.

For the select with colors, I just used a background color on each individual option.

For highlighting the selected input field, I used input:focus
