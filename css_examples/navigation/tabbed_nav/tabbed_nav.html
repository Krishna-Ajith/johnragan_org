Creating a Tabbed Navigation Menu Using a List

On the HTML side, you basically have a div (with id of navigation) which contains a normal ul with li elements. The one that is currently selected should the 'selected' class applied. The tab consists of a left image and right image of the tabs.

The main parts of the CSS are as follows:

  1. Set the left and right of the body to a margin of 10% for spacing
  2. The #navigation div should be floated left with a width of 100%.
  3. For the ul, set the list-style to none to get rid of the bullets
  4. Float the li left, with a background image of the left part of the tab image, with no-repeat
  5. For the li a, float it left as well, with a display of block to stretch the selection area out. Set its image to right part of the tab image, at the right and top with no-repeat and a text-decoration of none
  6. For the li with the selected li, set the left-image to be the left active one.
  7. For the li a with the selected li, set the right-image to be the right active one.
