Creating a Navigation Menu Using a List

On the HTML side, you basically have a div (with id of navigation) which contains a normal ul with li elements.

The main parts of the CSS are as follows:

  1. For the li under #navigation, the display is inline to go horizontal.
  2. The text-decoration has a value of none.
  3. Everything else is basically colors, spacing, and borders.
