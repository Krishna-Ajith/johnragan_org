Various Applications of text-decoration

  • This style (text-decoration : none;) removes underlines from links with underline
  • This style (text-decoration : overline;) adds an overline for links with overline
  • This style (text-decoration : line-through;) makes a line-through for links with line-through
  • This style (text-decoration : blink;) adds blinking links with blink
  • This style combines overline and line-through links

text-decoration can also be applied to general text as well
