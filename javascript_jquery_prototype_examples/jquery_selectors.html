CSS Selectors

Basic CSS Selectors

The following are the basic CSS selectors. If you want to see them in action, try jquery_events.html

  • $('p') - all paragraphs
  • $('.foo') - all entities with class 'foo'
  • $('#bar') - all entities with an id of bar

Nested CSS Selectors

This paragraph has a class of "para1" and is contained within a div. Thus, the following code causes it to be highlighted:

$('div .para1').addClass('highlight');
