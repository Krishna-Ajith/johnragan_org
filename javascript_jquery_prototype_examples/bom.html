Timers and Intervals

You can also set things to happen on an interval. The following will set a repeating interval of 10 seconds to make an alert come up. You set it with gInterval = setInterval(function_to_invoke_name, 10000); and clear it with clearInterval(gInterval); You can also inline as follows: var id = setInterval(function() { alert('boo'); }, 2000);
