Modifying Styles

This paragraph's CSS styles are set by javascript dynamically. Here are some that will be applied:

  • style.border = "1px solid red"
  • style.fontWeight = 'bold' (applied to this li only). Note that CSS font-weight (underscore illegal in JS) becomes font_weight, and so forth.
  • You can also modify attributes regardless of if they were initially set or not. Here we use align = "right" on that paragraph.
  • You can also directly assign id: id = 'further'

Creating

You can create with createElement() and createTextNode(). The following statements are used to create a new element, which we insert next.

  • var myp = document.createElement('p');
  • myp.innerHTML = 'yet another';
  • myp.style.border = '2px dotted blue'
  • document.getElementById('container').appendChild(myp);
